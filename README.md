# Sonic Heroes Game - C++ OOP Implementation

A fan-made Sonic Heroes game developed in C++ using Object-Oriented Programming principles, featuring custom mechanics, character control systems, and gameplay elements inspired by the original game.

## 🎮 Project Overview

This project is an educational implementation of the classic Sonic Heroes game mechanics using modern C++ with OOP design patterns. It demonstrates game development concepts including entity management, physics simulation, and character control systems.

## ✨ Features

- **Multi-Character System** - Control different characters with unique abilities
- **Object-Oriented Design** - Clean architecture with well-structured classes and inheritance
- **Custom Physics Engine** - Gravity, collision detection, and movement mechanics
- **Character Control** - Smooth character movement and interaction systems
- **Game Mechanics** - Custom gameplay elements inspired by Sonic Heroes

## 🛠️ Technology Stack

- **Language**: C++ (98.5% of codebase)
- **Architecture**: Object-Oriented Programming (OOP)
- **Build System**: CMake (recommended)
- **Platform**: Cross-platform

## 📋 Prerequisites

Before building this project, ensure you have:

- C++11 or later compiler (GCC, Clang, or MSVC)
- CMake 3.10+
- Git
- A graphics library (e.g., SDL2, SFML) - check project dependencies

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Abubakar123421/sonic_heroes_game_oop_cpp.git
cd sonic_heroes_game_oop_cpp
```

### Build Instructions

```bash
# Create build directory
mkdir build
cd build

# Generate build files
cmake ..

# Build the project
cmake --build .

# Run the game
./sonic_heroes_game
```
